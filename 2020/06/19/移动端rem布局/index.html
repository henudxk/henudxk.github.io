移动端rem布局
 2020-06-19 /  Sean

rem(font size of root element)即根元素(html)的字体大小，是一个相对单位

100vw代表视口的宽度，如果设计稿的宽度是750，直接用100vw / 750，那么1rem 就相等于1px，可以直接对着设计稿把px换成rem就可以了，还要给body设置一个默认字体，这样页面元素不设置字体的时候才不会去使用html的字体
